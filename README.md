## Orderbook Implementation

This repository contains my implementation of an orderbook for low-latency trading using Websocket API from Deribit.

You can 
1.Place
2.Cancel 
3.Edit orders
4.Fetch open order 
5.Fetch your orderbook by instrument or currency
6.Get ticker data for a currency or instrument
and much more!

Currrently this runs with a CLI interface.
